🌲 里山資本主義

貨幣中心から地域循環へ - 森・水・人の関係性を再起動する新しい経済

🎯 核心メッセージ

貨幣中心の成長依存を脱し、森・水・人の身近な資源と関係性を再起動させる「里山資本主義」で、生活コストを下げながら地域を稼ぐ循環をつくれる。

📊 問題の現状

  • 人口減・需要縮小で規模の経済が失速
  • 地域から資金が流出する脆弱性
  • 外部依存が家計と自治体を圧迫
  • 単一産業依存のリスク増大

🌱 里山資本主義の解決策

  • 木質バイオマスで熱・電力の地産地活
  • 空き家・遊休地などスラック資源活用
  • ナリワイ複線化でリスク分散
  • 支出削減による可処分所得増
  • 域内循環で付加価値を地域に留める

🌲 里山資本主義 地域循環の再起動

貨幣に依存しすぎず、森・水・土と人の関係性を資本として活用し、生活コストを下げつつ地域で稼ぐ仕組み。環境・経済・福祉を同時に回す。

💰 マネー資本主義 外部依存の脆弱性

貨幣利益の最大化を目的に、集中・規模の経済と外部依存を進める経済様式。人口減と需要縮小期には脆弱化する。

⚡ 分散型エネルギー 地産地活の電熱

木質バイオマス、太陽熱、小水力など地域スケールの発熱・発電。送配電ロスと価格変動リスクを抑える。

🔥 木質バイオマス 森を燃やして森を守る

間伐材や端材を燃料として熱・電力を得る技術と産業。林業整備と雇用創出を両立する。

🍅 地産地消 域内循環

地域で生産した食・エネルギー・サービスを地域内で消費し、付加価値を域内に残す仕組み。

🔀 ナリワイの複線化 小さく多く稼ぐ

単一の大企業雇用に依存せず、小さな仕事を複数持ちリスク分散と関係性を広げる生計戦略。

🔗 概念間の関係性と因果関係

💰 経済循環の流れ

  • 里山資本主義 → 生活コスト低下 → 可処分所得増
  • 分散型エネルギー → 外部調達費削減 → 域内付加価値増
  • 地産地消 → 物流距離短縮 → CO2/コスト減
  • 外部依存縮小 → 資金流出減 → 地域内再投資

🌱 環境と経済の好循環

  • 木質バイオマス × 林業整備 → 森林健全化
  • スラック資源活用 → 初期投資圧縮
  • コモンズ再生 → 信頼資本蓄積
  • ナリワイ複線化 → 所得分散 → 家計耐性向上

💡 支出削減の逆転発想

「お金を稼ぐより先に、地域から出ていく支出を減らせば可処分所得は増える。」

→ 成長が鈍い環境でも再現性高く家計と地域を強くする逆転の発想

🔥 熱エネルギーの実用性

「熱の自給は電力より導入が容易で、地域経済効果が大きい。」

→ 段階投資の起点として失敗確率を下げ、早期に便益を出せる実務的示唆

🌲 里山の価値再定義

「里山は不要不急ではなく、暮らしと安全保障を支えるインフラである。」

→ 環境保全をコストではなく生存基盤と捉え直すパラダイム転換

⚠️ 単一依存のリスク

「一つの大企業に頼る地域は、一つの故障点を抱えるシステムと同じだ。」

→ ナリワイ複線化と産業多様化の必要性を直感的に示す比喩

🤝 見えない資本

「眠る資源を動かす知恵と関係性こそ、地方の見えない資本である。」

→ 有形資産の再活用と社会関係資本の重要性を統合して示す
