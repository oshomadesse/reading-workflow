🎯 核心的メッセージ

貧困は単なる資金不足ではなく、情報不足・制度不信・行動バイアス・市場の欠陥が絡み合う現象である。現場に根ざした厳密な検証（とくにRCT）で仮説を分解し、低コストの設計変更と適切なインセンティブで大きな改善を実現できる。

🔬 ランダム化比較試験（RCT）

介入の効果を無作為化で識別する実験手法。因果推論の精度を高め、政策の有効性を検証可能にする。

検証

⏰ 現在バイアス

目先のコスト・快楽を過大評価し、将来の利益への投資を先送りする傾向。貯蓄やワクチン接種の遅れを招く。

先延ばし

🔒 コミットメント装置

将来の自分を縛る仕組み（貯蓄ロック等）で、先延ばしを減らし望ましい行動を促す設計。

縛り

💰 価格のしきい値効果

ごく小さな自己負担でも需要を急減させる非線形反応。無料化や小額補助が利用率を劇的に高める。

閾値

🤝 情報の質と信頼

正確で理解しやすい情報と供給者への信頼が行動の鍵。誤情報や腐敗は需要と成果を弱める。

信頼

🏦 マイクロファイナンス

小口融資で起業や消費を支える仕組み。ただし平均的効果は限定的で、貯蓄・保険等の補完が不可欠。

小口金融

📋 論理展開の流れ

  • 問題提起：貧困は単純な"資金不足"では説明できない
  • 生活領域の実証1：栄養と保健の微調整効果
  • 生活領域の実証2：教育の利益可視化と設計
  • 家庭内意思決定：社会的要因の影響
  • リスク管理：保険不在の歪み

🔗 概念間の関係性

  • 貧困 → 認知資源逼迫 → 現在バイアス強化
  • 微小コスト → 利用率急落 → 介入効果消失
  • 誤情報・低品質 → 不信 → 需要減
  • 保険欠如 → リスク回避 → 所得上限固定
  • エビデンス → 有効施策特定 → 学習サイクル

🎯 政策設計のアプローチ

"貧困は一つの大問題ではなく、多くの小問題の束だ"
介入を細分化し実証的に解くべきという設計思想を示す
"わずかな価格や手間の差が行動を劇的に変える"
しきい値効果への感度が政策の成否を決める
"尊厳のない援助は機能しない"
信頼と受益者視点が実装の前提条件

📊 重要な発見

  • 価格をわずかに下げるだけで利用率が飛躍的向上
  • 情報提供のタイミングと方法が成果を左右
  • マイクロファイナンスは万能薬ではない
  • 自営業の多くは"選べない自営"で生産性が低い
  • 保険の欠如がリスク回避行動を強化
