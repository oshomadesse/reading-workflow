  • 共通善を対話で形づくる
  • 公平に理由を示し合う市民的徳が必要

🎯 功利主義

"役に立てば善い"

幸福の総量で判断
多数の幸せを重視

🗽 リバタリアニズム

"触るな、私の自由"

個人の自由と所有権
国家介入を最小限に

📜 カント的義務論

"義務を守れ"

動機と規則を重視
人を手段にするな

🏛️ アリストテレス徳論

"ふさわしさの正義"

目的に応じた配分
徳を育てる社会

🤝 共通善（公共善）

  • 社会全体として望ましい善
  • 個人の好みの単純な足し算ではない
  • 対話で内容を創造する価値
  • 市民的徳の育成が前提

💰 市場と道徳の限界

  • お金で取引すると価値が損なわれる領域
  • 尊厳や友情は売買に向かない
  • 市場の拡張は便利だが危険も
  • 非市場の価値を保護する必要

📝 同意・契約の限界

  • 形式上の合意だけでは不十分
  • 圧力や格差が自由な選択を阻害
  • 正当化には背景条件の検討が必要
  • 真の自由意志の確保が課題

🎓 アファーマティブ・アクション

  • 歴史的不利を補う優遇策
  • 「実力（メリト）」の定義が争点
  • 公平への迂回路として機能
  • 目的論的視点が重要

⚔️ 対立関係

  • 功利主義 ↔ 義務論：結果 vs 規則
  • リバタリアニズム ↔ 共通善：自由 vs 結束
  • 個人の権利 ↔ 社会の目的
  • 市場の効率 ↔ 道徳の価値

🔄 因果関係

  • 市場拡張 → 非市場価値の腐食リスク
  • 同意の限界 → 自由交換神話の修正
  • トロッコ問題 → 直感の可視化
  • 公共討議 → 共通善の明確化

🌉 橋渡し

  • アリストテレス → 配分基準の提供
  • アファーマティブ・アクション → メリト再定義
  • 市民的徳 → 制度と人格の連結
  • 対話の習慣 → 理論と実践の統合
