🌸 沈黙の春

化学農薬が生態系と人間に与える影響を科学的に解明した環境保護の古典

🎯 核心メッセージ

  • 化学農薬の無自覚な大量使用は危険
  • 生態系と人間の健康を同時に損なう
  • 予防原則で方針転換が必要
  • 自然のしくみを活かす害虫管理へ

📊 主な問題

  • DDTなどが土・水・空気に残留
  • 食物網を通じて生物濃縮
  • 野鳥の減少と人の病気リスク
  • 企業・行政の安全軽視

💡 提案される解決策

  • 予防原則の採用
  • 統合的害虫管理（IPM）
  • 透明な規制と市民参加
  • 自然との共生重視

🔄 農薬問題の悪循環

🚁 広域散布の農薬 → 環境中の残留性増加
⬆️ 残留性増加 → 生物濃縮 → 上位捕食者への毒性リスク増大
🔄 単一化学依存 → 害虫の抵抗性進化 → 使用量増加
🌐 食物網ダメージ → 生態系サービス低下 → 農業生産性悪化

🧪 農薬（殺虫剤・除草剤）

害虫や雑草を殺すための化学物質。DDTは代表例で、効き目が長く残る点が問題になった。

化学で駆除

⏰ 残留性

化学物質が土や水、体内に長く残る性質。時間がたっても分解されにくい。

長く残る

📈 生物濃縮

汚染物質が食物連鎖を上がるほど体内で濃くなる現象。上位の生物ほど影響が大きい。

上ほど濃く

🕸️ 食物網

生き物同士の「食べる・食べられる」の網の目。汚染は網全体に広がる。

つながる食

🌍 生態系

生物と環境が相互作用するまとまり。小さな変化が全体に波及する。

つながりの場

🛡️ 予防原則

害が疑われる段階で使用を控える考え方。確実な証明を待たずリスクを避ける。

疑わしき回避

🤝 統合的害虫管理（IPM）

天敵の利用、作付け工夫、物理的防除などを組み合わせ、化学に依存しない方法。

賢く組み合わせ

💪 抵抗性

薬に慣れる
