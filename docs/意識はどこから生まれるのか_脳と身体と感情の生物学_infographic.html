🧬 生命の基盤

ホメオスタシス
生命調整
生体が内部環境を安定に保つ自動的な調整機構で、意識の土台となる動機づけと価値の源泉
体内受容
内側感覚
内臓や血流など身体内部の状態を検知する感覚モード。感情の素材を提供

💫 情動と感情

情動
行動プログラム
脳が引き起こす生得的な反応プログラム（表情・自律反応・行動）。意識化を要さない
感情
感じ
情動がもたらす身体状態の"感じ"としての経験。意識の質感を形成する

🎭 自己の三層

プロト自己
基盤自己
脳幹・島・体性感覚皮質などに広がる、身体状態の無意識マップ群。自己の最下層
コア意識（コア自己）
今ここ意識
対象との相互作用の瞬間に、自己と対象の関係が立ち上がる短時間の意識
自伝的自己
物語自己
記憶・言語・社会的知識を統合し、時間的連続性をもつ自己表象

🧠 神経基盤

脳幹
生命中枢
覚醒・原初的感情・生命維持の要で、意識の最低限の条件を支える
島皮質
内感覚皮質
体内受容情報の中核的皮質部位で、"感じ"の生成に寄与
収束-発散ゾーン
統合結節
多領域から情報が集約（収束）し再展開（発散）される結節点。再表象と統合を担う

🔗 概念間の関係性

ホメオスタシス → 情動（自動反応） → 感情（経験） → 価値付与 → 意思決定
体内受容マッピング（脳幹・島・体性感覚） → 感情の質感（フィール） → コア意識の成立
プロト自己 + 対象との相互作用 → コア自己 → 記憶統合 → 自伝的自己（拡張意識）
収束-発散ゾーン → 多感覚統合と再表象 → 自我の一貫性・安定性

⚠️ 病変からの証拠

  • 脳幹損傷 → 覚醒・感情障害 → 意識障害
  • 高次連合野のみの損傷では意識は部分的に保持
  • 前頭葉損傷 → 価値付与の障害 → 意思決定困難
  • 島皮質損傷 → 体内感覚の異常 → 感情認識の変化
  • 皮質だけでは意識は十分でなく脳幹と身体が不可欠
