過剰消費

消費主義

欲望の連鎖を刺激し続けることで経済成長を維持する価値観。時間や自然を犠牲にしても「もっと」を追う。

簡素さ

質素/節度（ソブリエダ）

必要十分にとどめる生き方。持たないことで束縛を減らし、自由と持続可能性を高める。

可処分人生

自由と時間

労働や消費に拘束されない可処分時間と自己決定の余地。真の豊かさの中核資源。

少欲知足

幸福の再定義

所有量ではなく、関係性・自由・意味に基づく充足。少欲知足の倫理。

限界内成長

持続可能な発展

地球の環境収容力の範囲で人間の福祉を高める発展。世代間公平を含む。

市場の神

市場至上主義

市場原理が社会のあらゆる領域を支配する状態。人間と自然が手段化される。

共に生きる

連帯と公共善

個人の利得を越え、共同で問題を解決する倫理と制度。脆弱な者の保護を含む。

心の貧富

貧困の再定義

欠乏の量ではなく、満足できない心の状態を貧しさとみなす視点。

地球の限界

自然の有限性

地球資源と吸収能力には上限があるという前提。無限成長と論理的に両立しない。

歯止め

政治の役割

市場に倫理的・環境的制約を課し、公共善と長期利益を代表する機能。

診断：現状批判
消費主義と市場至上主義が時間・自由・自然を侵食していると批判

豊かさと貧しさの再定義（所有量→満足・自由）
4
規範提案

制度提案

6
エトス：実証

7
結語：呼びかけ
地球の有限性を直視し、個人と社会の両輪で価値観の転換を促す

🔄 概念間の因果関係

  • 消費主義 → 労働時間増大/借金 → 自由と時間の喪失 → 幸福低下
  • 市場至上主義 → 自然と人間の手段化 → 持続可能性の損壊
  • 自然の有限性 → 成長の限界 → 発展の再設計の必要
  • 質素/節度 → 消費圧低下 → 可処分時間増加 → 自由と幸福の回復
  • 連帯と公共善 → 集合的ルール形成 → 持続可能な発展の実装
  • 政治の役割強化 → 市場に倫理的制約 → 長期的福祉の最大化
